  1. еҲӣе»әдёҖдёӘADе·ҘзЁӢпјҢеҜ№еә”PrjPCBж–Үд»¶пјҢеҲӣе»әеҺҹзҗҶеӣҫе’ҢPCBж–Үд»¶пјҢеҜ№еә”SchDocж–Үд»¶е’ҢPcbDocж–Үд»¶гҖӮ

  2. еңЁеҺҹзҗҶеӣҫж–Үд»¶дёӯе®ҢжҲҗеҺҹзҗҶеӣҫзҡ„и®ҫи®ЎпјҢеңЁе®ҢжҲҗи®ҫи®Ўд№ӢеҗҺпјҢеҸҜд»ҘйҖҡиҝҮADиҮӘеёҰзҡ„жӣҙж–°еҠҹиғҪпјҢе°ҶеҺҹзҗҶеӣҫдёҠзҡ„еҺҹд»¶жӣҙж–°еҲ°PCBж–Үд»¶дёҠпјҢиҝҷдёҖеҠҹиғҪзҡ„дҪҚзҪ®еңЁвҖқDesignвҖқ -> вҖңUpdate PCB Document \.PcbDocвҖқгҖӮ

  3. еҪ“еҺҹзҗҶеӣҫдёҠзҡ„и®ҫи®Ўе…ЁйғЁе®ҢжҲҗд№ӢеҗҺпјҢж—ўеҸҜеңЁPCBдёҠиҝӣиЎҢеёғеұҖе’ҢеёғзәҝдәҶгҖӮе…ғеҷЁд»¶еёғеұҖж—¶еә”иҜҘжіЁж„ҸпјҢе…ғеҷЁд»¶дёҺе…ғеҷЁд»¶д№Ӣй—ҙзҡ„еёғеұҖеә”иҜҘйҒөеҫӘе°ұиҝ‘еҺҹеҲҷгҖӮ

  4. е…ғеҷЁд»¶еёғеұҖе®ҢжҲҗд№ӢеҗҺе°ұеҸҜд»ҘиҝӣиЎҢеёғзәҝдәҶгҖӮеёғзәҝеҸҜд»ҘдҪҝз”ЁиҮӘеҠЁеёғзәҝпјҢд№ҹеҸҜд»ҘдҪҝз”ЁжүӢеҠЁеёғзәҝгҖӮ

  5. еңЁеёғзәҝеӨ§жҰӮе®ҢжҲҗд№ӢеҗҺе°ұеҸҜд»Ҙж №жҚ®е®һйҷ…зҡ„еёғеұҖжӣҙж”№еҗҲйҖӮзҡ„PCBжқҝеӨ§е°ҸдәҶпјҢеңЁViewдёӯеҲҮжҚўеҲ°Board Planning ModeпјҢеңЁеҪ“еүҚи§Ҷеӣҫз»ҳеҲ¶PCBзҡ„еӨ§е°ҸеҚіеҸҜгҖӮ

  6. ж”№еҘҪPCBзҡ„еӨ§е°Ҹд№ӢеҗҺпјҢеҲҮжҚўеҲ°Keep-out Layerз»ҳеҲ¶PCBзҡ„еӨ–иҫ№жЎҶпјҢйңҖиҰҒжіЁж„Ҹзҡ„жҳҜиҝҷдёҖжӯҘйңҖиҰҒе…ҲеҲҮжҚўеӣһ2D Layout ModeпјҢз»ҳеҲ¶еӨ–иҫ№жЎҶиҰҒдҪҝз”ЁPlace Keepout Trackз»ҳеҲ¶пјҢз»ҳеҲ¶е®ҢжҲҗеҗҺпјҢйҖүдёӯиҫ№жЎҶпјҢйҖҡиҝҮDesign -> Board Shape -> Define from selected objectзЎ®е®ҡиҫ№жЎҶзҡ„еӨ§е°ҸгҖӮ

  7. еңЁPCBжқҝзҡ„еӣӣдёӘи§’иҗҪж”ҫзҪ®е®үиЈ…еӯ”пјҢйҖҡеёёи®ҫзҪ®3mmеҶ…еҫ„пјҢ5mmеӨ–еҫ„гҖӮ

  8. ж·»еҠ ж»ҙжіӘпјҢеңЁTool -> Teardropдёӯж·»еҠ ж»ҙжіӘгҖӮйңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢж»ҙжіӘдёҚиғҪйҮҚеӨҚж·»еҠ пјҢеңЁж·»еҠ дәҶж»ҙжіӘд№ӢеҗҺпјҢеҰӮжһңиҰҒдҝ®ж”№жҹҗж №еҜјзәҝпјҢйңҖиҰҒз§»йҷӨе…ЁйғЁж»ҙжіӘпјҢдҝ®ж”№е®ҢжҲҗеҗҺеҶҚйҮҚж–°ж·»еҠ ж»ҙжіӘгҖӮ

  9. жңҖеҗҺж·»еҠ иҰҶй“ңпјҢеңЁPlace -> Polygen PourиҸңеҚ•дёӯж·»еҠ иҰҶй“ңгҖӮ

  10. еҚ•еҮ»Report жЈҖжҹҘPCBзҡ„и®ҫи®ЎпјҢе°Ҷи®ҫи®ЎжҠҘе‘Ҡдёӯзҡ„й—®йўҳе…ЁйғЁдҝ®ж”№е®ҢжҲҗеҚіеҸҜгҖӮ
